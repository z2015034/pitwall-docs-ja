# 利用状況

## 利用状況とは
Pitwallを利用していると様々なデータが蓄積されます。以下のようなデータを見ることができます。

- 推定ワークロードの節約
- Estimated cost saved
- 合計実行シナリオ数
- 最も実行されたシナリオ
- シナリオタイプの傾向

## 使用方法

初期画面
はじめて利用する場合、このような画面になります。

### 過去データ表示期間オプション
画面の右上のプルダウン矢印で、過去データの表示期間を選ぶことができます。
- 過去24時間
- 過去7日間
- 過去30日間
- 過去365日間
